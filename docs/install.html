安装

SWANPHP.0 是基于 PHP 5.3 以上开发的，全部应用命名空间所以安装非常简单，只需要将 sf1 目录拷贝到项目目录中，在引用的地方将 SWANPHP 的入口文件 swanphp.php 包含进去即可。

SWANPHP 在加载 swanphp.php 的时候就自动将 swan 命名空间名注册了, 所以可以用 \swan\xxxx 调用 SWANPHP 的 LIB.

例如：

项目引用目录结构如下：

[shell]
|-- index.php  // 项目入口文件
|-- lib
|   |-- Makefile
|   |-- Makefile.ini
|   |-- exception
|   |-- geshi
|   |-- github
|   |-- markdown
|   |-- sf  // SWANPHP 类库
|   |   |-- demo
|   |   |-- inc
|   |   |-- swan  // SWANPHP 核心库文件
|   |   |-- swanphp.php  // SWANPHP 库入口文件
|   |   |-- tests
|   |   `-- ventor
|   `-- ui

index.php 文件调用 SWANPHP 库需要如下方法调用

[php]
<?php
define('PATH_SWWEB_BASE', dirname(__FILE__));
    define('PATH_SWWEB_LIB', PATH_SWWEB_BASE . '/lib/');
// 引用 SWANPHP 入口文件
require_once PATH_SWWEB_LIB . 'sf/swanphp.php';
/*
..........
调用逻辑
*/

许可协议

SWANPHP 是在 MIT 许可下授权的。这表示你可以在严格保留版权信息的前提下自由的修改、分发和重发布源代码。你也可以自由地将 SWANPHP 包含在任何商业的或封闭源代码的应用程序中。
